# YouTube Clone

This is a YouTube clone built using React, Tailwind CSS, Redux Toolkit, Node.js, Express.js, and MongoDB with Mongoose.

## Technologies Used

- React
- Tailwind CSS
- React Router
- Redux Toolkit
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
